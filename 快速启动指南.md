# 快速启动指南

## 🚀 三步启动项目

### 第一步: 安装依赖

```bash
npm install
```

这将安装所有必需的依赖包,包括:
- Vue 3
- Vite
- Vant 4
- Vue Router
- Pinia
- Axios

### 第二步: 启动开发服务器

```bash
npm run dev
```

启动后会显示:
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
```

### 第三步: 访问应用

在浏览器中打开: http://localhost:5173

| test | 123456 | 测试用户 | 启用 |

## 🎯 完整使用流程

1. **登录**: 使用测试账号登录
2. **浏览菜单**: 查看商家公告和菜品分类
3. **添加商品**: 点击"+"按钮添加商品到购物车
4. **查看购物车**: 点击底部购物车栏进入购物车页面
5. **下单**: 点击"立即下单"提交订单
6. **查看结果**: 订单成功页面显示订单信息

## 🔧 常见问题

### Q1: npm install失败
**解决方案**:
- 尝试清除npm缓存: `npm cache clean --force`
- 使用淘宝镜像: `npm install --registry=https://registry.npmmirror.com`

### Q2: 登录失败提示"账号或密码错误"
**原因**:
- 飞书多维表格中没有对应的用户数据
- 用户状态不是"启用"

**解决方案**:
- 检查用户表数据
- 确保status字段值为"启用"

### Q3: 菜品列表为空
**原因**:
- 菜品表中没有数据
- 菜品状态不是"上架"
- 分类关联错误

**解决方案**:
- 添加菜品数据
- 确保status字段值为"上架"
- 正确关联分类

### Q4: 下单后订单表没有数据
**原因**:
- 飞书API权限不足
- 表ID配置错误

**解决方案**:
- 检查飞书应用权限配置
- 验证.env文件中的Table ID是否正确

3. 配置环境变量 (复制.env内容)

2. 在Netlify中导入项目
3. 构建命令: `npm run build`
4. 发布目录: `dist`
5. 配置环境变量
