## 🛠️ 技术实现细节

### 前端架构
```
Vue 3 Composition API
  ├─ Vite 5 (构建工具)
  ├─ Vant 4 (UI组件库)
  ├─ Vue Router 4 (路由管理)
  ├─ Pinia (状态管理)
  └─ Axios (HTTP客户端)
```

### 状态管理
- **购物车Store** (`stores/cart.js`)
  - 商品列表管理
  - 总数量/总金额计算
  - 增删改商品操作

- **用户Store** (`stores/user.js`)
  - 用户信息管理
  - 登录状态持久化
  - LocalStorage同步

### 飞书API集成
- **认证方式**: tenant_access_token
- **API封装**: `api/feishu.js`
- **支持的操作**:
  - 用户登录验证
  - 获取商家公告
  - 获取菜品分类
  - 获取菜品列表
  - 创建订单记录

## 🚀 使用流程

### 步骤1: 安装依赖
```bash
npm install
```

### 步骤2: 添加测试数据
参考 `手动添加测试数据指南.md` 在飞书多维表格中添加数据。

**最少数据要求**:
- 用户表: 1个用户
- 商家公告表: 1条公告
- 菜品分类表: 2个分类
- 菜品表: 5道菜品(需关联分类)

### 步骤3: 启动项目
```bash
npm run dev
```

### 步骤4: 访问测试
1. 访问 http://localhost:5173
2. 使用测试账号登录: test / 123456
3. 浏览菜单,添加商品到购物车
4. 进入购物车,调整数量
5. 点击立即下单,提交订单
6. 查看订单成功页面

## ⚠️ 注意事项

### 1. 数据初始化
- 必须先在飞书多维表格中添加测试数据
- 菜品必须关联分类才能正常显示
- 所有启用状态的字段必须设置正确

### 2. API权限
- 确保飞书应用有正确的API权限
- App ID和Secret必须配置正确
- 网络请求需要通过Vite代理

### 3. 浏览器兼容性
- 推荐使用Chrome浏览器
- 移动端建议使用Chrome或Safari
- 需要支持ES6+语法

### 4. 开发环境
- Node.js版本: v16+
- npm版本: v8+
- 推荐使用VS Code编辑器
